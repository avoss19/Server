SolidWorks with Legos

This page is to show my knowledge of SolidWorks through a project that we had to complete for my school's engineering course. Also all the files show in the images above can be downloaded from solidworks.andrewvoss.org. This project we had to recreate Lego parts in SolidWorks and also create parts that are able to work with Legos. Through creating the parts I also learned how to create some parts complex parts that can be resized through SolidWorks's use of equations. Another thing we had to learn was how to use the assembly in SolidWorks, which allows you to put parts that you have created together. Through this project, we also got trained to use our school's laser cutter.

The main purpose of this project was to train our class how to use SolidWorks so that we would be able to work on our school's robot that participates in the RoboCup competition. I feel that this project was important due to it giving me the skills to create just about anything from custom parts, prototypes, or finished products.
